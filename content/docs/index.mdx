# B2B Agency Boilerplate

Добро пожаловать в документацию нашего WordPress Boilerplate! Этот репозиторий представляет собой "Единый источник истины" (Single Source of Truth) для всей команды. 

<Callout type="info">
  **Главная цель Boilerplate:** дать возможность быстро стартовать новые проекты без написания базовых сущностей, при этом не создавая проблем с обновлением ядра (Git Upstream).
</Callout>

Здесь вы найдете всю необходимую информацию: от базовых принципов гибридной архитектуры до работы с CSS токенами и Gutenberg блоками.

## Основные принципы

1. **Святое ядро** — мы не удаляем и не изменяем файлы в `mu-plugins/agency-b2b-core/`. Все настройки происходят через *Feature Toggles* и фильтры.
2. **Гибридный контент** — страницы и услуги верстаются через Gutenberg, а жесткие структуры (кейсы, товары) — через Carbon Fields и классические шаблоны.
3. **Изоляция визуального слоя** — тема (`agency-starter-theme`) отвечает только за рендер данных и стили. Бизнес-логики в теме быть не должно!
